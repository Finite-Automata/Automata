Non-Deterministic Finite Automata

Non-Deterministic Finite Automata (NFA) - from the word non-deterministic, has a principle where it can be in, and transition to, "multiple states" at once depending on the given input. In NFA, there might be several possible final states that could result from reading a given string of inputs; as long as at least one of them is a final state, then the string is accepted.

Now that we know a NFA may transition to multiple state at a time, then it is obvious that the transition function of a NFA is different from DFA which can be defined as δ: Q x ∑→ 2 ^ Q. This means that the cross product of an element of set Q(states) and an element of set ∑(inputs) will have a transition to the element of power set of Q, not just from the elements of set Q itself.

Power set means "the set of all subsets of a given set", for example we have Q = {A,B,C}, the Power set of Q or P(Q) = {{∅},{A},{B},{C},{A,B},{A,C},{B,C},{A,B,C}}.

The definition of Deterministic Finite Automata is a 5-tuple (Q,Σ,I,F,δ) where:

  • Q: finite set of States.
  • ∑: finite set of Inputs.
  • q: the Initial/Starting State.
  • F: finite set of Final/Accepting States.
  • δ: Transition Function.

Graphical Representation of NFA

  1. The states are represented by vertices (circles).
  2. The arc or arrow labeled with input/s.
  3. The initial state is marked with plain arrow.
  4. The final state/s are marked by double circle.

2.) Transition Function of NFA
a) ∑ x Q -> 2 ^ ∑
b) Q x ∑ -> 2 ^ Q
c) ∑ x Q -> 2 ^ Q
d) Q x ∑ -> 2 ^ ∑

VIEW ANSWER

Answer: B
NFA's transition function is the cross product of Q(State) and ∑(Input) which will have a result of the power set of Q or 2^Q.

3.) The Principle of NFA
a) Multiple State at a time
b) One State at a time
c) none of the above

VIEW ANSWER

Answer: A Explanation: A NFA can be in and Transition to MULTIPLE STATES at a time.

4.) When will NFA accept a string?

b) At least one of the paths end in final state

Answer: B Explanation: The NFA accepts string if at least one transition end in the FINAL STATE.
